🕐 技嘉出勤時間追蹤器

Microsoft Edge 擴充套件 - 智能追蹤您的上下班時間

📋 專案簡介

這是一個專為技嘉員工設計的 Microsoft Edge 瀏覽器擴充套件，能夠自動追蹤您的上下班時間，並根據公司的彈性上班制度智能計算預計下班時間。

🔐 安全登入

直接連接技嘉 EIP 系統，使用您的公司帳號安全登入

⏰ 智能計算

根據彈性上班制度自動計算預計下班時間

📊 即時顯示

即時顯示今日出勤狀況和剩餘工作時間

💾 記憶功能

可選擇記住登入資訊，方便日常使用

⚙️ 彈性上班制度規則

  • 彈性上班時間：8:30 - 9:30
  • 8:30 上班 → 17:45 下班
  • 8:30-9:30 之間上班：固定工作 9小時15分鐘
  • 9:30 上班 → 18:45 下班
  • 超過 9:30 上班 → 18:45 下班

🚀 安裝指南

方法一：從 Microsoft Edge 商店安裝（推薦）

  1. 開啟 Microsoft Edge 瀏覽器
  2. 前往 Edge 附加元件商店
  3. 搜尋「技嘉出勤時間追蹤器」
  4. 點擊「取得」進行安裝
  5. 安裝完成後，點擊瀏覽器工具列上的擴充套件圖示

方法二：手動安裝開發版本

  1. 下載專案檔案到本機
  2. 開啟 Microsoft Edge，前往 edge://extensions/
  3. 開啟右上角的「開發人員模式」
  4. 點擊「載入解壓縮」按鈕
  5. 選擇下載的專案資料夾
  6. 擴充套件安裝完成！

📖 使用說明

首次使用

  1. 點擊瀏覽器工具列上的擴充套件圖示
  2. 輸入您的技嘉帳號（格式：gigabyte\your.username）
  3. 輸入密碼
  4. 可選擇「記住登入資訊」以便下次快速登入
  5. 點擊「登入」按鈕

日常使用

登入成功後，擴充套件會自動顯示：

  • 今日日期
  • 上班打卡時間
  • 下班打卡時間（如已打卡）
  • 根據彈性制度計算的預計下班時間
  • 剩餘工作時間
⚠️ 注意事項：
  • 此擴充套件僅供技嘉員工使用
  • 需要有效的技嘉 EIP 系統帳號
  • 登入資訊會安全儲存在本機，不會傳送到第三方伺服器
  • 建議定期更新擴充套件以獲得最新功能
